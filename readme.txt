La solution visual studio se trouve dans le dossier Tetris.
Le fichiers sources se trouvent dans le dossier Tetris/Tetris/src et Tetris/Tetris/headers.
Les dépendences de la SDL (.dll, .lib, .h) se trouvent dans le dossier Dependencies/SDL2.
La présentation du projet est à la racine du projet.

Il y a des commentaires dans les headers de chaque classe pour savoir à quoi elles servent
et comment les utiliser.

Lors de la distribution du programme, les fichiers .dll et le dossier
"ressources" : "Tetris/Tetris/ressources" doivent être dans le même dossier que l'exécutable.
